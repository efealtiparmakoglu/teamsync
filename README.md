# 💼 TeamSync

Gerçek zamanlı işbirliği platformu. Slack + Trello + Zoom bir arada.

## Özellikler
- 💬 Kanal tabanlı mesajlaşma
- 📹 Video konferans
- 📋 Proje yönetimi (Kanban)
- 📁 Dosya paylaşımı
- 📝 Not alma (Wiki)
- 🔔 Bildirimler
- 🤖 Bot entegrasyonu

## Teknolojiler
- React + TypeScript
- Node.js + Socket.io
- MongoDB
- WebRTC
- AWS

## Kurulum
```bash
npm install
npm run dev
```
